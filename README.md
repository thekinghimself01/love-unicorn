# Love Unicorn

A simple, romantic single-page web project with a themed "love unicorn" experience.

## Project Structure

- `index.html` — Main landing page.
- `letters.html` — Secondary page for love letters or messages.
- `script.js` — Client-side JavaScript for page interactions.
- `styles.css` — Styling for the site.
- `favicon.svg` — Site favicon.
- `images/` — Project image assets.
- `netlify.toml` — Netlify site configuration.
- `netlify/functions/check-pin.js` — Serverless function for PIN verification.

## Deployment

This project is configured for deployment on Netlify. The `netlify/functions/check-pin.js` file is used by the site for secure PIN checking.
